## Блокировка клиента
[GET /clients/{client_id}/block_status](https://github.com/yn96k/client-block-service/blob/main/client-block-service-v1.openapi.yaml)

### Входные параметры 
| Название |Тип параметра| Описание |
|:--|:---|:---|
|clientId|path|Идентификатор клиента|

### Алгоритм метода:
1. Запрашивает поля block_type, comment из таблицы client_block БД client:
   
```sql
SELECT client_id, block_type, blocked_at, comment
FROM client_block
WHERE client_id = {clientId из входных параметров}
```
1.1. Если запись отсутствует в БД - вернуть ошибку с кодом 404 <br>
2. Завершить работу алгоритма, вернув успешный ответ с кодом 200 и телом ответа согласно контракту:

|Поле|Что передается|
|:--|:--|
|clientId|client_id из БД client|
|blockType|block_type из БД client|
|blockedAt|blocked_at из БД client|
|comment|comment из БД client|
