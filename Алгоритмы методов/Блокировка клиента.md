## Блокировка клиента
[POST /clients/{client_id}/block](https://github.com/yn96k/client-block-service/blob/main/client-block-service-v1.openapi.yaml)

### Входные параметры 
| Название |Тип параметра| Описание |
|:--|:---|:---|
|client_id|path|Идентификатор клиента|
|block_type|body|Название причины блокировки|
|comment|body|Комментарий модератора|

### Алгоритм метода:
1. Обновляет поля block_type, comment в таблице client_block БД client:
   
```sql
UPDATE client_block
SET block_type = {block_type из входных параметров}
SET comment = {comment из входных параметров}
WHERE client_id = {client_id из входных параметров}
```
1.1. Если запись отсутствует в БД - вернуть ошибку с кодом 404 <br>
2. Завершить работу алгоритма, вернув успешный ответ с кодом 200.
